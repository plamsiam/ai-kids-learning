# AI สำหรับเด็ก - เรียนรู้ AI สนุกๆ 🤖

เว็บไซต์เรียนรู้เกี่ยวกับ AI สำหรับเด็ก ออกแบบมาให้น้องๆ หนูๆ เรียนรู้เกี่ยวกับ AI อย่างสนุกและปลอดภัย

## 🌟 ฟีเจอร์

- 📝 **ทดลองความรู้** - Quiz 8 คำถาม พร้อมสุ่มคำตอบ
- 🎮 **เกมทาย Emoji** - ให้ AI ทายอิโมจิ 70% ถูกต้อง
- 💬 **คุยกับ AI** - เชื่อมต่อ Google Gemini API (ฟรี)
- 📚 **เรียนรู้ AI** - เนื้อหาเข้าใจง่ายสำหรับเด็ก
- 🏆 **ระบบคะแนน** - บันทึกความคืบหน้าอัตโนมัติ
- ⚠️ **ข้อควรระวัง** - สอนการใช้ AI อย่างปลอดภัย

## 🚀 Deploy ด้วยตัวเอง

### Netlify (แนะนำ - ง่ายที่สุด)
1. ไปที่ https://netlify.com
2. สมัครสมาชิกฟรี
3. ลากโฟลเดอร์ `ai-kids-learning` วางใน Dashboard
4. ได้ URL ทันที!

### GitHub Pages
1. สร้าง repository ใหม่บน GitHub
2. อัพโหลดไฟล์ทั้งหมด
3. ไปที่ Settings > Pages > เลือก branch main
4. เข้าชมที่ `https://username.github.io/ai-kids-learning`

## 📁 โครงสร้างไฟล์

```
ai-kids-learning/
├── index.html              # หน้าแรก
├── what-is-ai.html         # AI คืออะไร?
├── ai-capabilities.html    # AI ทำอะไรได้บ้าง
├── quiz.html               # ทดลองความรู้
├── game.html               # เกมทาย Emoji
├── daily-life.html         # AI ในชีวิตประจำวัน
├── caution.html            # ข้อควรระวัง
├── css/
│   └── style.css           # สไตล์ทั้งหมด
└── js/
    └── app.js              # ฟังก์ชันทั้งหมด
```
